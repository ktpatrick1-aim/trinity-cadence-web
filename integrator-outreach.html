Pre-Send Scan Process (5 seconds)

Run this checklist before sending ANY outreach email, LinkedIn message, or marketing material:

  • Ctrl+F / Cmd+F for: L10, Rock, Scorecard, IDS, V/TO, GWC, People Analyzer, Accountability Chart, Core Focus, 10-Year, 3-Year, Quarterly Conversation
  • If ANY match is found: replace with the Trinity Cadence equivalent from the table above
  • If referencing a competitor by name: remove it (save competitor references for live demos only)
  • If describing what the product does: use generic terms ("weekly leadership meeting," "90-day priorities") or Trinity vocabulary (Huddle, Anchors, Pulse)
  • Read the email aloud. Does it sound like a peer sharing a tool, or a salesperson pitching? If the latter, rewrite.
